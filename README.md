# Awesome One Person Company [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of the best resources, tools, AI workflows, and templates for building a profitable one-person company.

A one-person company is a fully-fledged business — real revenue, real customers, real systems — run by a single founder with no employees. AI tooling and modern automation have made it possible to operate at a scale that used to require a whole team. This list collects the highest-signal resources for solopreneurs, solo founders, and indie hackers who want to build alone.
